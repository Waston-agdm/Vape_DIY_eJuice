# ⚡ DIY 果汁计算器 (Nimbus Lab)

这是由 **Nimbus Lab** 开发的私人调油辅助工具。

## 🌐 在线访问
网址：[点击此处访问 [GitHub Pages 链接](https://waston-agdm.github.io/Vape_DIY_eJuice/)]

## 🛠️ 主要功能
- **精准计算**：自动换算原液、VG、PG 及多成分香精配比，精确至 0.01ml。
- **本地存档**：配方数据仅存储于用户当前浏览器的本地缓存中，保护隐私。
- **内置预设**：集成参考配方库。
