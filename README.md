# ⚡ DIY 果汁计算器V2.0
这是一个专为电子烟 DIY 玩家设计的网页版计算器。基于 HTML5 和 Tailwind CSS 构建，支持高精度配比计算、香精配方库检索以及个人配方本地存储功能。

## 🌐 在线访问
**项目网址**：`https://waston-agdm.github.io/Vape_DIY_eJuice/`

---

## ✨ 核心功能
* **高精度配比**：支持原液浓度、PG/VG 比例自动换算，结果精确至小数点后两位。
* **官方配方库**：内置某品牌香精配方，支持关键词快速搜索。
* **私人收藏夹**：利用浏览器 LocalStorage 技术，支持在本地保存、加载或删除你的个人配方。
* **自适应设计**：完美适配手机端与电脑端，方便在调油现场实时操作。
* **溢出提醒**：当香精或原液占比超过设定比例时，系统会自动发出红色警告。

---

## 🛠️ 如何使用
1.  **成品设定**：设置你想要制作的总毫升数（ml）和目标尼古丁浓度（mg）。
2.  **调整比例**：滑动进度条选择你期望的 PG/VG 比例（如 50/50 或 40/60）。
3.  **原料输入**：
    * 输入你手头原液的实际浓度。
    * 点击“添加成分”录入香精、凉味剂或甜味剂的名称及百分比。
4.  **按图索骥**：右侧表格会实时显示你分别需要抽取多少毫升的 **原液、香精、VG 和 PG**。
5.  **保存配方**：点击“保存当前”，配方将永久保存在你当前设备的浏览器中。
